  • Create a boolean vector (True/False) "bool_cond" which checks whether
    "usd_to_jpy" is below 100
  • Generate a line plot with the "usd_to_jpy" series, label the axes and the title
  • Use the command plt.fill_between() as in Lecture 22 to create a shaded region
